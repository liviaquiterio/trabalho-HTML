How-to HTML e CSS

Tag HTML

A tag HTML de conteúdo é uma marca que define o conteúdo de um documento da respectiva linguagem. Ela envolve o texto e os elementos que aparecem no documento.

Tag body

A tag body é que vai definir o corpo de um site. Lá é que estarão as informações visíveis para o usuário. Tudo que você está lendo nesse momento, os links e imagens que estão vendo, estão dentro da tag.

Tag p

A tag "p" é utilizada para demarcar um parágrafo.

Tag h1, h2, h3, h4, h5, e h6

A tag H1 é a mais importante da página e deve ser atribuída ao título principal do conteúdo. As tags H2 são vinculadas aos subtítulos, as tags H3 a outros trechos de destaque (mas com menor relevância que as tags H2), e assim por diante. No entanto, raramente é necessário utilizar tags H4, H5 e H6.

Tag span

Na prática, a tag span serve para que você seja capaz de representar elementos visuais destacados de um todo e não representa nada por natureza.

Tag pre

é a tag utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo

Tag div

A tag div cria um bloco genérico para agrupar conteúdos.

Seletores CSS

Os Seletores definem quais elementos um conjunto de regras CSS se aplica.

Seletores por tag

Este seletor básico escolhe todos os elementos que correspondem ao nome fornecido.

Seletor por classe

Este jogador básico escolhe elementos com base no valor de seu atributo classe. Sintaxe: .nome-da-classe. Exemplo: .index irá corresponder a qualquer elemento que tenha o índice de classe (provavelmente definido por um atributo class="index", ou similar).

Seletor por ID

Este seletor básico escolhe nós com base no valor do atributo id. Deve existir apenas um elemento com o mesmo ID no mesmo documento.
